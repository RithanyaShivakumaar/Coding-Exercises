This repository contains solutions for two exercises as part of the 2024-25 coding assignment, focusing on design patterns and a mini-project.

Exercise 1: Design Patterns
Demonstrates six use cases across three categories:

Behavioral: Observer, Template Method
Creational: Singleton, Factory
Structural: Adapter, Decorator
Each use case showcases practical implementation and follows best practices for maintainability and scalability.

Exercise 2: Astronaut Daily Schedule Organizer
A console-based application that helps astronauts manage tasks. Key features:

CRUD operations for tasks with validation for task conflicts.
Uses Singleton, Factory, and Observer patterns.
Optimized for performance, with robust logging and error handling.
